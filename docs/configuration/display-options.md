# Display Options

Control how your metrics are visually presented in the Polystat panel. These options allow you to fine-tune the layout, appearance, and user experience.

## Available options

- **Customization Options** - Detailed settings for layout, sizing, text formatting, sorting, and tooltips
- **Layout Guide** - Best practices for organizing and displaying multiple metrics

Adjust these settings to create the perfect visual representation of your monitoring data.
