# Configuration

The Polystat panel provides comprehensive configuration options to customize how your metrics are displayed. This section covers all the essential settings you need to configure your panel.

## What you'll find here

- **Global Settings** - Panel-wide configuration including shapes, colors, and display modes
- **Display Options** - Customize layout, sizing, text, tooltips, and sorting
- **Thresholds** - Define state-based coloring using ranged thresholds

Each section provides detailed guidance on configuring specific aspects of the panel to match your monitoring needs.
