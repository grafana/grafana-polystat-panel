# Advanced Features

The Polystat panel includes powerful advanced features that enable sophisticated monitoring scenarios and enhanced interactivity.

## What you'll find here

- **Data Customization** - Fine-tune how specific metrics are displayed and formatted
- **Composites** - Combine multiple metrics into a single representation
- **Clickthrough URLs** - Add interactive navigation to your dashboards

These features unlock the full potential of the Polystat panel for complex monitoring environments.
